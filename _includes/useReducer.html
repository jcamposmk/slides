Hook useReducer

Es un hook de React para actualizar un estado interno por medio de una función llamada reducer.
  • Es uno de los hooks integrados por defecto en React
  • Muy similar a useState
  • Nos permite delegar la responsabilidad de la lógica de estado a alguien más
  • Nos permite mantener limpio el componente.
¿Cuándo usarlo?
  • Cuando se tiene un estado complejo, que implique multiple subvalores o propiedades interrelacionadas
  • Cuando sobre el estado se realizan muchas operaciones diferentes (existen varias formas de actualizar el estado)
  • Cuando el estado siguiente depende del anterior
Ejemplos de casos de uso:
  • Una llamada a una api
  • Una lista de tareas
  • Carrito de compra
Llamada a una api de planes
  • Hay que indicar un tipo de mensaje al usuario que indique la busqueda de los datos (loading: true)
  • Guardar los datos recibidos de la llamada (plans) e indicar que ya se resolvio la llamada (loading: false)
  • En caso de un error, indicar al componente que hubo un error (hasError) e indicar que ya se resolvio la llamada (loading: false)
Cada una de estas operaciones, representan una acción diferente que se puede realizar sobre el estado.
